## ✨ Kenmerken
* **Interactiviteit:** Deeltjes volgen de muis met een vertragingseffect.
* **Responsiviteit:** Het canvas past zich automatisch aan de venstergrootte aan.
* **Kleurenpalet:** Dynamische kleuren op basis van de snelheid van de muis.

## 🛠️ Gebruikte Technieken
* **p5.js Library:** Voor de core rendering en wiskunde.
* **HTML5 Canvas:** Voor de visuele output.
* **CSS3:** Voor de styling van de interface.
